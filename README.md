# Machine Learning Regression Projects

### 1. Insurance Charge Prediction
Sigorta ücretlerini tahmin etmek için çeşitli regresyon modellerinin uygulandığı proje.

**Dataset:**
https://www.kaggle.com/datasets/mirichoi0218/insurance

**Kullanılan Modeller:**
- Linear Regression
- Polynomial Regression
- Lasso / Ridge / ElasticNet
- LassoCV / RidgeCV / ElasticNetCV
- KNN Regressor
- Decision Tree Regressor
- SVR

**En İyi Sonuç:** Polynomial Regression (R² = 0.84)
